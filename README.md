## 프로젝트 소개

- Coffee Balance 은 사용자들의 커피 건강을 책임지는 앱 어플리케이션 입니다.
- 하루 커피 섭취량을 계산하여 어느정도 마셔도 되는지 확인할 수 있습니다.
- 내가 먹는 커피의 영양소 정보를 한눈에 확인하실 수 있습니다.
- 1주일, 한달 커피 섭취량을 확인하고 어제 먹었던 카페인 양과 오늘의 양을 비교해 어느정도 먹었는지 확인할 수 있습니다.
